# Collection of Mounting Clips
The BCOTI is meant to be very modular in regard to what devices it can be mounted on, this folder contains some ready-to-go designs.
I've you have a device or lens type that is not support yet, feel free to reach out to me.

## List of Clips

### PVS14_InfinityRing_Clip
- Uses an M2 screw to clamp on the infinity-ring (also called a D-Lock Ring).
- Requires 1x M2 insert, and an M2 screw with a length of 12mm or longer. The Side with the small X is the side you put the insert into.

### Baldur_InfinityRing_Clip
- Same as `PVS14_InfinityRing_Clip` only with a slightly larger ID of 38mm and 8mm length to support Baldur Housing.

### Old_Fenrir_InfinityRing_Clip
- Same as `PVS14_InfinityRing_Clip` only with a slightly larger ID of 40mm and 8mm length to support older version of the Fenrir Mono housing.

### ANVIS_Front_Lens_Clamp
- Confirmed to work with Anvis 6/9/9(4lobed) lenses.
- Requires 1x M2 insert, and an M2 screw with a length of 12mm or longer. The Side with the small X is the side you put the insert into.
